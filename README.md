# arch
Arch Linux setup

## local profile
```
alias pbcopy='xsel --clipboard --input'
alias pbpaste='xsel --clipboard --output'
```

## Basics
```
sudo pacman -S git
sudo pacman -S yaourt
sudo pacman -S openssh
```

## Usage
```
sudo pacman -S xsel
```

## Applications
```
yaourt -S google-chrome
```
